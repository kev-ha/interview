### Word Counter Application

Run the application with JRE 8 or greater.

Import this to your preferred IDE. The main class is `Application.java`. It takes input continuously. The word `exit` is the word to terminate the application.

The logic is contained in `logic` package, file name is `WordCounter.java`.

Example running the application with Spring Tool Suites. Note the output in the console window.
